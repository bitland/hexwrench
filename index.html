hexwrench by emonti

A gui hex editor based on ruby and wxwidgets (via wxruby)

hexwrench aims to be a scriptable hex editor which can be easily modified and extended using the ruby scripting language.

Here's a screenshot:

Dependencies

wxruby >= 0.2.0
wxirb is optional (integrates an IRB console)

Install

Gem Installation

    gem sources -a http://gems.github.com #(you only have to do this once)
    gem install emonti-hexwrench

Manual Installation

You can also download the tarball, zipball, or clone via git.

To generate the gem locally do:

  cd hexwrench
  rake gem:install

or ... you can install manually without rubygems:

  cd hexwrench
  cp -r hexwrench/lib/* /usr/lib/ruby/1.8/site_ruby/1.8 # or another ruby libdir
  cp bin/* ~/bin      # or wherever else in your PATH

Run this to generate docs with rdoc the same way the gem would have:

  rdoc --main README.rdoc README.rdoc lib
