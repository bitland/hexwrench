hexwrench by emonti

A portable graphical hex editor based on ruby and wxwidgets (via wxruby)

hexwrench aims to be a portable and scriptable hex editor which can be easily modified or extended using the ruby scripting language.

Documentation

The rdoc API documentation may or may not be of use to you.

Dependencies

wxruby >= 0.2.0
wxirb is optional (integrates an IRB console)

Install

Gem Installation

      gem sources -a http://gems.github.com #(you only have to do this once)
      gem install emonti-hexwrench

See below for download and manual build info.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/emonti/hexwrench

get the source code on GitHub : emonti/hexwrench

Manual Installation

To generate the gem locally do:

    cd hexwrench
    rake gem:install

or ... you can install manually without rubygems:

    cd hexwrench
    cp -r hexwrench/lib/* /usr/lib/ruby/1.8/site_ruby/1.8 # or another ruby libdir
    cp bin/* ~/bin      # or wherever else in your PATH

Run this to generate docs with rdoc the same way the gem would have:

    rdoc --main README.rdoc README.rdoc lib
