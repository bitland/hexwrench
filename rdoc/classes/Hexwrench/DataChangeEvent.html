Class Hexwrench::DataChangeEvent
In: lib/hexwrench/edit_window.rb
Parent: Wx::CommandEvent

The DataChangeEvent event is fired when a user makes any change to data in the editor window. It is used mostly by the parent window to trigger behaviours in various other UI elements when this happens.
