Module Hexwrench::UtilTextCtrl
In: lib/hexwrench/edit_frame.rb

This module is used to extend a regular Wx::TextCtrl text box to display greyed out text when idle which describes the control‘s purpose. As soon as focus is set on the control, the text disappears.

Methods

clear_error   do_default_text   do_error   init   on_kill_focus   on_set_focus  

Public Instance methods

clear_error()

Clears error text set by do_error and returns the textbox to its default text display.

do_default_text()

Returns text to its greyed-out default display

do_error()

Indicates an error in the text box by turning current text red.

init(opts={})

This init method is called after creation since we don‘t override ‘new’ in the XRC derived window element. Takes a hash of :name => value options. The only option is currently :default_text, which is the text to display when the control is idle. If :default_text is not specified, the control will use the initial value in the text box as default text.

on_kill_focus(evt)

Event handler for when focus is lost to this text control. Returns the text to its default display unless an error in input was flagged.

on_set_focus(evt)

Event handler for when focus is set to this text control. Clears default text in preparation for user input.
